# Hệ Thống Quản Lý Hải Sản

Nền tảng quản lý toàn diện cho cửa hàng bán hải sản tươi sống với tính năng:
- Quản lý sản phẩm & biến thể
- Quản lý kho hàng & hao hụt
- Xử lý đơn hàng & giao vận
- Bán hàng online & tại quầy (POS)
- Báo cáo & phân tích tài chính
- CRM & Marketing

## Stack Công Nghệ
- **Backend**: PHP (Laravel hoặc CodeIgniter)
- **Frontend**: HTML/CSS/JavaScript + Vue.js
- **Database**: MySQL
- **Admin Dashboard**: Vue.js 3
